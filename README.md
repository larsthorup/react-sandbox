Vanilla JS react implementation, inspired by
https://pomb.us/build-your-own-react/

- Avoid any compilation steps during development (Webpack, JSX, TypeScript)

```bash
npm install

npm start
npm run start:build

npm test
npm run test:build
```

Coverage report in `output/index.html`.
